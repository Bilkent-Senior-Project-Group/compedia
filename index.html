Project Overview

Project Goal

Create an intelligent platform that helps businesses find relevant partners through semantic search and AI-driven matching.

Technologies

Built using React, ASP.NET Core, LLMs (GPT/BERT), Vector Databases, and modern web technologies.

Academic Context

Developed as part of CS491/2 Senior Design Project course at Bilkent University.

Key Features

Semantic Search

Natural language processing using GPT and BERT models for context-aware search

Vector Database

Efficient storage and retrieval of semantic data using Milvus vector database

Project Management

Integrated tools for tracking collaborations and managing business relationships
